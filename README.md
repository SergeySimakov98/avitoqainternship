Для запуска тестов необходим python 3.14
1. Склонируйте репозиторий 
2. Перейдите в корневую директорию 
3. Создайте и активируйте виртуальное окружение (python -m venv venv, venv\Scripts\Activate) 
4. Установите зависимости (pip install -r requirements.txt)
5. Запустите тесты командой (pytest .\test_avito.py)
